# PalindromeCheckerApp
* **Goal: Determine if a string is a palindrome by reversing it.
* Logic:
* Use a for loop to iterate characters in reverse order.
* Build the reversed string using the + operator.
* Compare the original and reversed strings using .equals().
* Technical Details
* String Immutability: Every modification creates a new object in memory.
* Content Comparison: Use equals() to check actual text, not memory references.
* Data Structure: Uses the String class.
* Quick Commands
* Compile: javac UseCase3PalindromeCheckerApp.java.
* Run: java UseCase3PalindromeCheckerApp.**
